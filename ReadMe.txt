scriptIn.js позволяет единоразово определить область <script> кода и писать в нее html код с прямыми вставками js кода через оператор '{{' '}}'.

Для использования в HTML необходимо создать блок <script type="text/components">, и внутри можно писать HTML код, совмещая его с js-кодом, посредством оператора '{{' '}}'.
